## 摘要
多模态思维链（CoT）推理要求大型视觉语言模型构建推理轨迹，将感知落地（perceptual grounding）与多步推理交织在一起。然而，现有的具有可验证奖励的强化学习（RLVR）方法通常以粗粒度优化推理，将 CoT 轨迹视为统一整体，而未能区分其不同程度的视觉落地。在这项工作中，我们对多模态推理轨迹进行了 Token 级分析，结果表明成功的推理具有结构化的 Token 动态特征，能够同时反映感知落地和探索性推理。基于此分析，我们提出了感知-探索策略优化（PEPO），该方法从隐藏状态相似性中推导出感知先验，并通过平滑门控机制将其与 Token 熵相结合，从而产生 Token 级的优势值。PEPO 与现有的 RLVR 框架（如 GRPO 和 DAPO）无缝集成，既不需要额外的监督，也不需要辅助分支。在多种多模态基准测试上的广泛实验证明，PEPO 在几何推理、视觉落地、视觉谜题解决和少样本分类等任务中，相比强强化学习基线取得了持续且稳健的提升，同时保持了稳定的训练动态。

## Abstract
Multimodal Chain-of-Thought (CoT) reasoning requires large vision-language models to construct reasoning trajectories that interleave perceptual grounding with multi-step inference. However, existing Reinforcement Learning with Verifiable Rewards (RLVR) methods typically optimize reasoning at a coarse granularity, treating CoT uniformly without distinguishing their varying degrees of visual grounding. In this work, we conduct a token-level analysis of multimodal reasoning trajectories and show that successful reasoning is characterized by structured token dynamics reflecting both perceptual grounding and exploratory inference. Building upon this analysis, we propose Perception-Exploration Policy Optimization (PEPO), which derives a perception prior from hidden state similarity and integrates it with token entropy through a smooth gating mechanism to produce token-level advantages. PEPO integrates seamlessly with existing RLVR frameworks such as GRPO and DAPO, requiring neither additional supervision nor auxiliary branches. Extensive experiments across diverse multimodal benchmarks demonstrate consistent and robust improvements over strong RL baselines, spanning geometry reasoning, visual grounding, visual puzzle solving, and few-shot classification, while maintaining stable training dynamics. *   **研究动机**：现有的多模态大模型（LVLM）在进行思维链（CoT）推理时，通常采用具有可验证奖励的强化学习（RLVR，如 GRPO）。然而，这些方法大多基于**序列级（Sequence-level）**的监督，将整个推理轨迹视为统一整体。
*   **核心问题**：序列级优化粒度过粗，无法区分推理过程中不同 Token 的贡献。作者发现，成功的推理高度依赖于“视觉落地（Perceptual Grounding）”和“探索性推理（Exploration）”的结合，而现有方法忽视了这种 Token 级别的动态差异。
*   **整体含义**：本论文旨在通过 Token 级别的细粒度优化，增强模型在推理过程中对视觉信息的利用能力及对不确定推理路径的探索能力。

#### 2. 方法论
*   **核心思想**：提出 **PEPO (Perception-Exploration Policy Optimization)**，通过计算 Token 级别的感知先验和探索信号，重新分配序列优势函数（Advantage），实现细粒度策略更新。
*   **关键技术细节**：
    1.  **感知建模（Perception Modeling）**：利用模型内部隐藏状态（Hidden States）的余弦相似度，计算每个文本 Token 与所有视觉 Token 之间的关联度（Visual Similarity, VS），作为感知落地的指标。
    2.  **探索建模（Exploration Modeling）**：利用输出 Logits 计算 Token 级别的熵（Entropy, H），识别推理中的关键决策点或不确定步骤。
    3.  **感知-探索融合**：通过一个**平滑门控机制（Smooth Gating Mechanism）**将归一化后的 VS 和 H 结合，生成 Token 权重 $w_t$。该机制确保感知信号占主导，同时允许熵信号对视觉相关的 Token 进行调节。
    4.  **Token 级优势分配**：将原始序列级优势 $A$ 调整为 $A_t = [(1-\lambda) + \lambda w_t] A$，其中 $\lambda$ 随训练过程动态增加。
*   **算法集成**：PEPO 可无缝集成至 GRPO（PEPO-G）或 DAPO（PEPO-D）框架中，无需辅助模型或额外的监督信号。
