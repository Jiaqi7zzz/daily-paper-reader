## 摘要
近期在具有可验证奖励的强化学习（RLVR）方面的进展为语言模型的自我提升提供了一条切实可行的路径，但现有方法面临一个关键的权衡：内源性自博弈（endogenous self-play）可能会随着迭代而产生漂移，而基于语料库的方法则依赖于精心策划的数据环境。我们提出了 WIST，一个用于领域定向推理提升的基于 Web 基础的迭代自博弈树框架，它直接从开放网络中学习，无需任何预先安排的领域语料库。WIST 增量地扩展领域树以进行探索，并检索和清洗路径一致的 Web 语料库，以构建一个可控的训练环境。随后，它利用可验证奖励进行“挑战者-求解者”（Challenger--Solver）自博弈，并将可学习性信号反馈以更新节点后验，通过自适应课程指导后续探索。在四个骨干模型上，WIST 相比基础模型均有一致的提升，且通常优于纯内源性自我演化和基于语料库的自博弈基准模型，总体增益分别达到 +9.8（Qwen3-4B-Base）和 +9.7（OctoThinker-8B）。WIST 还具有领域可控性，在医学领域将 Qwen3-8B-Base 提升了 +14.79，在 PhyBench 上将 Qwen3-4B-Base 提升了 +5.28。消融实验进一步证实了 WIST 关键组件对于稳定开放网络学习的重要性。我们的代码可在 https://github.com/lfy-123/WIST 获取。

## Abstract
Recent progress in reinforcement learning with verifiable rewards (RLVR) offers a practical path to self-improvement of language models, but existing methods face a key trade-off: endogenous self-play can drift over iterations, while corpus-grounded approaches rely on curated data environments. We present \textbf{WIST}, a \textbf{W}eb-grounded \textbf{I}terative \textbf{S}elf-play \textbf{T}ree framework for domain-targeted reasoning improvement that learns directly from the open web without requiring any pre-arranged domain corpus. WIST incrementally expands a domain tree for exploration, and retrieves and cleans path-consistent web corpus to construct a controllable training environment. It then performs Challenger--Solver self-play with verifiable rewards, and feeds learnability signals back to update node posteriors and guide subsequent exploration through an adaptive curriculum. Across four backbones, WIST consistently improves over the base models and typically outperforms both purely endogenous self-evolution and corpus-grounded self-play baselines, with the Overall gains reaching \textbf{+9.8} (\textit{Qwen3-4B-Base}) and \textbf{+9.7} (\textit{OctoThinker-8B}). WIST is also domain-steerable, improving \textit{Qwen3-8B-Base} by \textbf{+14.79} in medicine and \textit{Qwen3-4B-Base} by \textbf{+5.28} on PhyBench. Ablations further confirm the importance of WIST's key components for stable open-web learning. Our Code is available at https://github.com/lfy-123/WIST.

### WIST：基于 Web 基础的迭代自博弈树领域定向推理提升总结

#### 1. 核心问题与整体含义（研究动机和背景）
论文关注大语言模型（LLM）的**自我提升（Self-improvement）**。目前主流的强化学习方法（如 RLVR）面临两难境地：
*   **内生自博弈（Endogenous Self-play）**：如 R-Zero，完全不依赖外部数据，但随着迭代增加，模型生成的任务容易产生“分布漂移”和质量退化。
*   **语料库驱动自博弈（Corpus-grounded Self-play）**：如 SPICE，依赖于预先整理的高质量领域语料库，这限制了其向新领域扩展的灵活性。

**WIST** 的核心动机是利用**开放互联网（Open Web）**作为无限且动态的知识源，通过结构化的探索机制，在无需预设领域语料的情况下，实现模型在特定领域（如数学、物理、医学）推理能力的持续进化。

#### 2. 方法论：核心思想与技术细节
WIST 构建了一个闭环系统，包含以下三个关键组件：
*   **动态领域树构建（Dynamic Domain Tree）**：
    *   从用户指定的领域标签（如“数学”）开始，模型递归地将领域分解为子课题，直至叶子节点的“最小知识点”。
    *   引入 `unk` 占位符节点，当采样到该节点时触发树的增量扩展。
    *   利用 Wikipedia 标题匹配进行非叶子节点的合法性验证，防止幻觉概念污染树结构。
*   **Web 检索与语料库自动化建设**：
    *   根据树的路径生成查询词，从 Web 检索相关文档。
    *   实施多级清洗：URL 过滤、语义嵌入相似度筛选（确保文档与知识点匹配）、正文提取及去重。
*   **Web 基础的挑战者-求解者自博弈（Challenger-Solver Self-play）**：
    *   **Challenger（挑战者）**：基于检索到的文档生成 QA 对。
    *   **Solver（求解者）**：在不看文档的情况下回答问题。
    *   **奖励设计**：Solver 获得基于正确性的二元奖励；Challenger 获得基于 Solver 回答方差的“难度塑形奖励”（鼓励生成处于模型能力边界、具有可学习性的问题）。
    *   **优化算法**：采用 **Dr. GRPO** 算法进行策略优化。
*   **后验引导的自适应课程**：
    *   为每个树节点维护一个 **Beta 分布后验**，记录该路径产生“可学习实例”的概率。
    *   使用**汤普森采样（Thompson Sampling）**选择探索路径，并配合**滑动窗口**更新机制以应对模型能力提升带来的非平稳性。

#### 7. 优点：亮点总结
*   **无需人工标注语料**：真正实现了从原始 Web 到领域专家能力的自动化进化。
*   **动态自适应**：通过 Beta 后验和汤普森采样，模型能自动聚焦于自己的“薄弱环节”，形成天然的课程学习。
*   **结构化探索**：领域树将模糊的 Web 搜索转化为结构化的知识点覆盖，有效缓解了自博弈中的语义漂移问题。

#### 8. 不足与局限
*   **验证器依赖**：WIST 仍然高度依赖**可验证奖励（RLVR）**。在法律、人文等缺乏客观、自动化判分标准的领域，该框架的适用性受限。
*   **Web 噪声敏感度**：尽管有清洗流程，但 Web 数据的质量波动仍会影响训练稳定性，尤其是对于判别能力较弱的小规模模型。
*   **检索成本**：相比纯内生方法，WIST 在训练过程中涉及频繁的 Web 检索和文档处理，增加了系统复杂度和推理外的计算开销。
